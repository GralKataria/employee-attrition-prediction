# Employee Attrition Prediction

## Overview
This project predicts whether an employee will leave the company using machine learning.

## Objective
To identify key factors influencing employee attrition and build a predictive model.

## Dataset
IBM HR Analytics Dataset

## Process
- Data Cleaning
- Exploratory Data Analysis
- Feature Encoding
- Model Training
- Model Evaluation

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest Classifier

## Accuracy
~87%

## Key Insights
- Overtime increases attrition
- Income affects retention
- Job role impacts attrition

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
